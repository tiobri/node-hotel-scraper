> Web-Scraper that search vacancies in a hotel website and returns it to an API.

## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

```sh
npm install
```

## Configuration

```sh
mv .env.example .env
```

## Usage

```sh
npm run start
```

The application will run on port 3000. If you want to use other port change variable PORT inside .env file.
